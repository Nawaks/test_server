## visreg

### Prerequisites

* [Node](http://nodejs.org/)'s [NPM](https://npmjs.org/) (just install Node v4, it ships with NPM)
* [Gulp](http://gulpjs.com/) (`npm install -g gulp`)

### Retrieve dependencies

    npm install

### NPM Scripts

    npm run build
    npm run dev
    npm run pm2start
    npm run pm2stop
    npm run pm2kill
